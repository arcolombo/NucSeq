Description:
  Convert a sorted BAM file into a cff file.

Usage:
  bam_to_cff.sh  [OPTIONS]  -- BAM_FILE

Options:
  -h--help                Help.
  -o--outdir              Output directory. Default: $PWD

Examples:
  bam_to_cff.sh -d cff -- paired_end.bam 

Dependancies:
  samtools
  bedtools

Upstream:
  bam_sort.sh

Notes:
  - BAM file needs to be sorted beforehand.
