Description:
    Applies a kernel smoother to the input. 

Usage:
    kernel_smoother.sh  [options]  -- CFF_FILE

Options:
    -h|--help           Help.
    -d|--outdir         Output directory. Default: $PWD.
    -t|--threads        Number of threads. Default: 2.
    -k|--kernel         Type of kernel used: "normal","box","epanech","biweigth","triweight". Default: "normal".
    -b|--bandwidth      Bandwidth of the kernel. Default: 75.

Examples:
    kernel_smoother.sh -d out -k normal -b 75 -t 8 -- contig_1.fa 

Dependancies:
    R{KernSmooth}
    R{doParallel}
    R{foreach}
    bedtools

Upstream:

    bedgraph_to_midpoints
