Description:
    Cross a cff file with a reference gff file.

Usage:
    peaks_in_reference.sh  [options]  -- CFF_FILE GFF_FILE

Options:
    -h|--help           Help.
    -d|--outdir         Output directory. Default: $PWD.
    -t|--threads        Threads used. Default: 2.

Examples:
    peaks_in_reference.sh -d out -t 8 -- r4s1.cff.gz reference.gff

Dependancies:
    bedtools

Upstream:
    peak_calling.sh
