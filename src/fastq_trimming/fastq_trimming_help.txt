Description:
  Trim reads in a .fastq file.

Usage:
	fastqTrim.sh  [options]  -- FASTQFILE

Options:
  -h|--help           Help
  -d|--outdir         Output directory
  -f|--outfilename    Outfile name
  -l|--readLength     Final read length. Default: '50'

Examples:               
  fastqTrim.sh -l 40 -- testRound1_R1.fastq.gz
