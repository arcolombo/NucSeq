Description:
  Convert a sorted BAM file into a compressed bedGraph file.

Usage:
  bam_to_bedgraph.sh  [OPTIONS]  -- BAM_FILE

Options:
  -h--help                Help.
  -o--outdir              Output directory. Default: $PWD

Examples:
  bam_to_bedgraph.sh -d bedgraph -- paired_end.bam 

Dependancies:
  samtools
  bedtools

Upstream:
  bam_sort.sh

Notes:
  - BAM file needs to be sorted beforehand.
